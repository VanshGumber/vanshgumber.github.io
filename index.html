RAG · NLP · SYSTEMS
Retrieval-Augmented Generation (RAG) Document Q&A System
An end-to-end RAG pipeline that enables real-time question answering over user-uploaded documents using dense semantic retrieval and LLM-based reasoning. The system combines paragraph-aware chunking with adaptive FAISS indexing to scale efficiently, while LLM-driven query rewriting improves retrieval robustness under vocabulary mismatch. A custom RAGAS-inspired evaluation framework provides label-free diagnostics across retrieval and generation quality, enabling systematic optimization of grounded responses.
Python FAISS RAG Ollama
Computer Vision · Research · In Progress
Adaptive Multi-Model Routing for Object Detection
A condition-aware routing pipeline that classifies input images and dynamically selects specialised YOLOv8 detectors based on predicted degradation severity — fog, rain, low-light. Severity is quantified by measuring baseline detector failure against panoptic segmentation masks, enabling data-driven routing under distribution shift. Integrates Grad-CAM attribution to verify attention on meaningful degradation cues.
Python YOLOv8 PyTorch OpenCV Grad-CAM In Progress

ML lifecycle pipeline that monitors model drift and triggers automated retraining. Evolving datasets generated via thermochemical equilibrium simulation create physically meaningful concept shifts. Automated drift detection using performance deviation thresholds — not fixed schedules. Sliding-window retraining with versioned artifacts and lifecycle logging mirrors production ML maintenance.
